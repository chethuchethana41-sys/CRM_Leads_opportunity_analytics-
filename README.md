# 📊  CRM_Leads_opportunity_analytics-
CRM Sales Analytics Dashboard using Excel,Python,MYSQL,Power BI to analyze opportunities, leads, conversion rates, and revenue insights.
🚀 Project Overview

This project analyzes CRM sales and lead data to generate insights about sales opportunities, lead conversions, and revenue performance.

The project follows a complete data analytics workflow, starting from data extraction and preprocessing to visualization and business insights.

The goal is to help organizations optimize their sales pipeline, identify high-performing industries, and improve lead conversion strategies.

⚙️ Data Analytics Workflow

1️⃣ Data Collection – Excel
CRM sales and lead datasets were initially collected and stored in Excel files.

2️⃣ Data Storage & Querying – MySQL
Data was imported into a MySQL database to perform structured queries and extract relevant datasets.

3️⃣ Data Cleaning & Processing – Python
Using Python (Pandas) for:

    Data cleaning
    
    Handling missing values
    
    Data transformation
    
    Preparing structured datasets for analysis

4️⃣ Data Visualization – Power BI
Interactive dashboards were built in Power BI to visualize KPIs and business insights.
