# CRM Sales Analytics (Leads → Revenue)

A data analytics project focused on identifying where sales are lost in the funnel and how revenue can be improved using CRM data.

---

## ⚡ What I Did

- Analyzed **10K+ CRM records** (leads, opportunities, accounts)
- Built end-to-end pipeline: **data cleaning → SQL analysis → Power BI dashboard**
- Tracked **lead-to-conversion funnel performance**
- Identified **revenue concentration & sales inefficiencies**

---

## 📊 Key Results

- 📉 Found ~20% drop in mid-funnel stages (major conversion leak)
- 💰 Top regions driving ~35–40% of total revenue
- 📈 Clear visibility into **conversion rates & sales performance**
- ⚡ Automated reporting using Power BI (reduced manual effort)

## 🧠 Insights

- Biggest loss happens in **qualification → proposal stage**
- Revenue is **highly concentrated (few regions dominate)**
- Sales performance is **inconsistent across reps**
- High-value deals convert better but occur less frequently
